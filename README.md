# BLDC Motor Thrust Testing Rig

## Overview
A test rig designed to evaluate the performance of BLDC motor–propeller systems by measuring thrust, RPM, and electrical parameters under controlled conditions. The system enables accurate analysis and comparison of different propulsion setups.

## Objectives
- Measure thrust generated by BLDC motor and propeller  
- Monitor RPM, voltage, and current in real time  
- Analyze performance and efficiency  
- Provide a reliable testing platform for UAV propulsion systems  

## System Architecture
- **BLDC Motor + Propeller:** Generates thrust  
- **ESC (Electronic Speed Controller):** Controls motor speed  
- **Load Cell + HX711:** Measures thrust force  
- **Microcontroller (ESP32 / Arduino):** Data acquisition and processing  
- **Power Source:** Battery or power supply  
- **Display / Serial Monitor:** Outputs real-time data  

## Working Principle
The BLDC motor is driven by an ESC, which controls its speed. As the propeller generates thrust, the force is measured using a load cell. The HX711 amplifier converts the load cell signal into digital data, which is processed by the microcontroller. RPM and electrical parameters are recorded simultaneously to evaluate system performance.

## Features
- Real-time thrust measurement  
- RPM monitoring  
- Voltage and current measurement  
- Data logging capability (optional)  
- Modular design for testing different motor–propeller combinations
